## ⚙️ Engineering Principles I Ship With

- **Webhook idempotency first.** Payment-adjacent systems deduplicate before they process. Always.
- **Multi-tenancy at the DB layer.** Application-level scoping is not enough. Global query scopes on every tenant-owned model.
- **Queues over synchronous processing.** If it can fail silently, it belongs in a queue with a dead-letter alert.
- **Security is architecture, not a feature.** HMAC verification, rate limiting, and input validation are in Week 1, not Week 7.
